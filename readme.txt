== Description ==
Heaven Brew is a sleek e-commerce theme for cafes, offering a smooth shopping experience for premium coffee products.
Elevate your coffee store with Heaven Brew, a modern theme designed for seamless shopping and stylish product displays.
Heaven Brew is a premium theme for coffee enthusiasts, featuring intuitive navigation and an elegant layout.
Showcase premium coffee with Heaven Brew, a beautifully designed theme for cafes and online coffee shops.

== Installation ==
1. Download the Heaven Brew theme zip file.
2. Navigate to your website's admin panel.
3. Go to Themes > Upload Theme and select the downloaded file.
4. Click Install and then Activate the theme.
5. Now your ready to customize your website.
